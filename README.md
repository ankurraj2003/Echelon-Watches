# Echelon Watches  
🔗 Live Demo: www.echelonwatches.shop

## 📌 Description
Echelon Watches is a luxury watch e-commerce platform offering a seamless shopping experience with an intuitive UI, powerful sorting features, and an admin panel for easy management.

## 🚀 Features
- 🏬 **E-commerce functionality** - Browse, add to cart, and purchase luxury watches.
- 🎛 **Advanced Sorting & Filtering** - Sort watches by brand, price, and style.
- 🎨 **Smooth & Modern UI** - A visually appealing interface with a responsive design.
- 🔑 **Admin Panel** - Manage products, orders, and users effortlessly.
- 🔍 **Search Functionality** - Find watches quickly with a powerful search.
- 💳 **Secure Payments** - Integration with secure payment gateways Razorpay and Stripe.
- 📦 **Order Tracking** - Keep users updated with real-time order tracking.

## 🛠 Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Token), Bcrypt
- **Payment Integration:** Stripe, Razorpay

## 🛠 Installation
```bash
# Clone the repository
git clone https://github.com/ankurraj007/Echelon-Watches.git
cd Echelon-Watches

# Install dependencies
npm install

# Start the application
npm start
```

## 📖 Usage
1. Browse and select your favorite luxury watch.
2. Use sorting and filtering to find the perfect watch.
3. Add items to the cart and proceed to checkout.
4. Admin users can log in to manage products and orders.
